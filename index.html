About Me

I am a Software Engineer with 2+ years of professional experience, specializing in backend development and API engineering. I build secure, scalable systems using Laravel, Node.js, and modern JavaScript frameworks, with a strong focus on clean architecture, performance, and reliability.


My work spans across developing REST APIs for mobile and web apps, implementing JWT/Sanctum authentication, building real-time chat and game systems, handling complex booking & discount logic, and integrating external services such as Firebase, AWS S3, and various third-party APIs.


I have hands-on experience working with queues, jobs, caching, database optimization, and production deployments on Nginx and Google Cloud. I also work with React, Core UI, and Material UI to create modern, responsive interfaces for dashboards and admin panels.


I am passionate about writing clean code, automating processes, managing GitHub workflows, and contributing to reliable, long-term software solutions. Outside of coding, I enjoy mathematics, playing chess, and exploring competitive video games.
